## 🧠 My Investigation Approach

When an alert triggers, I follow structured analysis:

1. Validate alert authenticity  
2. Check related logs and correlate events  
3. Reconstruct timeline  
4. Map behavior to MITRE ATT&CK  
5. Assess potential impact  
6. Decide escalation level  
7. Document clearly  

I strongly believe that context determines severity — not just the alert itself.
